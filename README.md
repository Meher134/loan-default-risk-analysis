# Loan Default Risk Analysis

## Project Overview
This project analyzes borrower financial and demographic characteristics to understand factors influencing loan default risk. The analysis combines data cleaning, exploratory analysis, machine learning modeling, SQL analytics, and an interactive Tableau dashboard.

## Tools Used
- Python (Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib)
- SQLite
- Tableau
- Jupyter Notebook

## Dataset
The dataset contains borrower financial attributes such as income, credit score, loan amount, and employment history.

## Analysis Workflow
1. Data Cleaning and Preprocessing
2. Feature Engineering (Debt-to-Income ratio, Loan Burden)
3. Exploratory Data Analysis
4. Machine Learning Models
   - Logistic Regression
   - Random Forest
5. SQL Queries
6. Tableau Dashboard Visualization

## Key Insights
- Only ~8% of borrowers default, indicating a highly imbalanced dataset.
- Younger borrowers show slightly higher default rates.
- Higher debt-to-income ratios correlate with increased default risk.
- External credit scores are strong predictors of repayment behavior.
